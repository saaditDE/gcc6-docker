# Kali Docker gcc6 compiler
![preview gcc6-docker](peek2022.gif)
# Installation
- do ./install.sh

# How to use?
- put file to compile into current directory
- do ./buildcontainer.sh
- do ./compileascript.sh
- use the compiled binary in current directory
